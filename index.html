CSV File Converter

This application allows for the importing of a .csv file following the template structure provided below to be used to convert client data into JSON or FileZilla Site Settings XML for easy creation of FileZilla site's within the Site Manager via FileZilla's built in Import function.

CSV Example

The top row should remain unchanged, this provides the application with the key values necessary to properly parse the data. Each row underneath is where you should provide the required values.

name host port username password directory
Site Name Site Host Site Port Number Site User Name Site Password Remote Directory

How to Use:

  1. Enter a filename (ex: client_name) this will append the provided name to the downloaded file (ie: client_name_site_import.xml) optional
  2. Select options for exported data. (ex: Remove passwords from uploaded data before conversion)
  3. Select your CSV file that contains the data you want converted.
  4. Download the file format you need.
